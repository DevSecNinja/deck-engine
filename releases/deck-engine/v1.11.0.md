# @deckio/deck-engine v1.11.0

## Cross-platform `deck-inspect` skill

The `deck-inspect` skill now works across all environments:

- **VS Code**: Uses Playwright browser tools (`screenshot_page`, `open_browser_page`) as before.
- **GitHub Copilot CLI** (Windows, macOS, Linux): Uses the new `capture-slide.mjs` script which auto-detects Puppeteer or falls back to the launcher capture API.
- **Containers (AKS)**: Puppeteer + Chrome are already available in the launcher Docker image; the capture script finds them automatically.

New file: `scripts/capture-slide.mjs` — standalone cross-platform capture utility.

## Removed `deck-generate-image` skill

The `deck-generate-image` skill and `scripts/generate-image.mjs` have been permanently removed. The skill was previously disabled from provisioning in v1.10.0 and is now fully deleted.
