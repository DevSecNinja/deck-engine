# @deckio/deck-engine v1.9.5

**Released:** 2026-03-19

## What changed

### Fixed
- Harden PowerPoint and PDF export capture by waiting for slide assets and fonts before rasterizing each slide.
- Preserve live slide dimensions during export instead of forcing a fixed 1920×1080 capture size, which fixes clipped tags and layout drift on non-16:9 or dense slides.
- Add export-mode document flags so decks can apply export-safe styling during capture.
