# create-deckio v0.3.1

Patch release fixing two scaffolder issues.

## Bug fixes

- **Icon prompt garbled on many terminals** — replaced the 4-byte `🎴` emoji default with `♠` (U+2660), a universally-renderable BMP character. The previous emoji displayed as `��` on terminals that couldn't handle supplementary Unicode.
- **Stale engine version fallback** — bumped the bundled `engineVersion` from `^1.11.0` to `^1.13.0` so users installing from npm get the latest compatible deck-engine.
